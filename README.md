# Pizza Sales SQL Project

## Overview
This project analyzes pizza sales data using SQL and answers 11 business questions related to orders, revenue, pizza performance, category contribution, and monthly sales trends.

## Dataset
The project uses the following files:
- orders.csv
- order_details.csv
- pizzas.csv
- pizza_types.csv

- pizza-sales-project-tushar.sql
- pizza-sales-dataset.sql

- pizza_types.csv

## SQL Questions Solved
1. What is the total number of orders placed?
2. What is the total revenue generated from pizza sales?
3. Which pizza has the highest price?
4. Which pizza size is ordered most frequently?
5. Which are the top 5 most ordered pizza names by quantity?
6. How are orders distributed by hour of the day?
7. Which 3 pizza names generated the highest revenue?
8. What is the percentage revenue contribution of each pizza category?
9. What is the total quantity of each pizza category ordered?
10. How many pizzas are present in each category?
11. What is the total revenue month-wise?
